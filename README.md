Список задач
Подключение к базе данных: Настроить application.properties.
Реализация сущности "Задание": Создать класс Task с полями: id, название, описание, статус, дата создания, дата окончания, is_done.
Реализация репозитория заданий: Создать репозиторий для Task с операциями CRUD (создать, Получить по id, обновить, удалить) и методом Найти все().
Логгирование: Добавить логгирование ошибок.
